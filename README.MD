# Data Analyst Agent API

An AI-powered data analysis API that can source, prepare, analyze, and visualize data using Large Language Models.

## Features

- Web scraping from Wikipedia and other sources
- CSV/Excel file processing
- Database querying (DuckDB, S3)
- Data visualization with regression analysis
- RESTful API with file upload support

## Setup

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Set your OpenAI API key in `.env`
4. Run the application: `python app.py`
